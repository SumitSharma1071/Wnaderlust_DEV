Wanderlust – Travel Listing Web App

Wanderlust is a full-stack web application that allows users to explore, create, and manage travel listings. It provides a platform where users can discover destinations, add their own listings, and interact with travel content.

Live Demo : https://wnaderlust-dev-2.onrender.com

Overview

Wanderlust is designed to simulate a real-world travel platform where users can browse listings, view details, and manage their own travel posts. The project focuses on building a scalable full-stack application with clean UI and efficient backend handling.
